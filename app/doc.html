Documentação do Sistema de Controle de Vagas

O sistema de controle de vagas desenvolvido é uma aplicação web utilizando HTML, CSS e JavaScript, com integração ao framework Electron para funcionalidades adicionais. Este sistema é projetado para gerenciar vagas de estacionamento, permitindo o registro, edição e finalização do estacionamento de veículos, bem como o cálculo automático de tarifas com base no tempo de permanência.

Funcionalidades Principais

  1. Registro e Gerenciamento de Vagas:

    Permite que os usuários visualizem as vagas de estacionamento, tanto ocupadas quanto disponíveis, e registrem novos veículos, incluindo informações como nome, placa, tipo, ano e horário de entrada.

  2. Armazenamento Local:

    Utiliza o localStorage para armazenar as informações dos veículos e vagas, garantindo a persistência dos dados mesmo após o fechamento da aplicação ou recarregamento da página.

  3. Visualização de Vagas:

    Exibe as vagas de estacionamento em uma tabela, mostrando o número da vaga, e indica se está ocupada ou disponível. Oferece a possibilidade de visualizar detalhes e editar as informações dos veículos.

  4. Cálculo de Tarifas:

    Calcula automaticamente a tarifa de estacionamento com base no tempo de permanência do veículo. O cálculo é feito considerando uma taxa fixa por hora e a diferença entre o horário de entrada e saída.

  5. Alertas e Feedback ao Usuário:

    Fornece alertas personalizados para orientar o usuário sobre a necessidade de inserir o horário de saída e confirmações de ações realizadas, como edição e finalização de estacionamento.

Estrutura da Aplicação

A aplicação é estruturada em um único arquivo HTML que define a estrutura da página. Utiliza folhas de estilo CSS para o design e scripts JavaScript para a lógica de funcionamento. Os principais componentes são:

  • Menu e Sidebar: Oferecem navegação para diferentes seções da aplicação, como Início, Clientes e Registro de Veículos.
  • Container de Vagas: Tabela que exibe as vagas de estacionamento, mostrando se estão ocupadas ou disponíveis.
  • Seção de Estatísticas: Mostra informações sobre o total de vagas, vagas disponíveis e vagas ocupadas.
  • Modais: Permitem a visualização e edição dos detalhes do veículo, bem como a inserção do horário de saída e cálculo da tarifa.
  • Caixa de Alerta: Exibe mensagens de alerta personalizadas para o usuário.

Implementação Técnica

O código JavaScript controla a lógica de negócios, manipulando o armazenamento local, renderização da tabela e cálculo de tarifas. Os principais métodos incluem:

  • renderGarage(): Renderiza a tabela de vagas com base nos dados armazenados e atualiza as estatísticas.
  • showDetailsModal(car): Exibe um modal com detalhes do veículo e opções para finalizar ou editar o estacionamento.
  • configureModalListeners(modal, finishBtn, editBtn, car): Configura os listeners para os botões dentro do modal de detalhes.
  • showEditModal(car): Exibe um modal para editar as informações do veículo e atualiza o armazenamento local ao salvar as alterações.
  • checkOut(car, exitTime): Calcula a tarifa com base no horário de entrada e saída, e mostra o resultado em um alerta personalizado.
  • showAlert(message): Exibe mensagens de alerta para o usuário.
  • convertPeriod(startTime, endTime): Calcula o período entre o horário de entrada e saída e converte-o em tarifa.

Conclusão

O sistema de controle de vagas oferece uma solução eficiente para gerenciar vagas de estacionamento, proporcionando aos usuários uma interface intuitiva para registrar, editar e finalizar o estacionamento de veículos. A integração com o localStorage permite a persistência dos dados, enquanto a lógica de cálculo de tarifas e alertas personalizados garante uma experiência de usuário aprimorada e automatizada.
