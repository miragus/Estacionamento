Documentação do Sistema de Estacionamento

O sistema de estacionamento desenvolvido é uma aplicação web utilizando tecnologias padrão como HTML, CSS e JavaScript, juntamente com o framework Electron para funcionalidades adicionais de desktop. Este sistema permite o registro de veículos em um estacionamento, monitorando seus horários de entrada e saída, calculando tarifas com base no tempo de permanência.

Funcionalidades Principais

  1. Registro de Veículos:

    Os usuários podem inserir o nome do veículo, a placa e o horário de entrada através de um formulário na interface.

  2. Armazenamento Local:

    Utiliza o localStorage do navegador ou a API equivalente do Electron para armazenar os registros de veículos. Isso permite que os dados persistam mesmo após o fechamento da aplicação.

  3. Listagem de Veículos:

    Os veículos registrados são exibidos em uma tabela, mostrando o número da vaga, nome do veículo, placa, horário de entrada e campos para inserção do horário de saída.

  4. Cálculo de Tarifas:

    Quando um veículo é marcado como pronto para sair (com a inserção do horário de saída), o sistema calcula automaticamente o tempo de permanência e a tarifa com base em uma taxa fixa por hora.

  5. Alertas e Feedback ao Usuário:

    O sistema fornece alertas para orientar o usuário, como avisos de campos obrigatórios não preenchidos ou vagas completamente ocupadas. Feedback visual é dado através de uma caixa de alerta personalizada para comunicar mensagens importantes ao usuário.

Estrutura da Aplicação

O sistema é estruturado em um único arquivo HTML que define a estrutura da página, vinculando folhas de estilo CSS para o design e scripts JavaScript para a lógica de funcionamento. Os principais componentes são:

  • Formulário de Registro: Permite a inserção de novos veículos com nome, placa e horário de entrada.
  • Tabela de Registro: Mostra os veículos registrados, incluindo campos para o horário de saída, permitindo ação de exclusão.
  • Caixa de Alerta: Utilizada para mensagens de alerta ao usuário, com opção para fechar após leitura.

Implementação Técnica

O código JavaScript controla a lógica de negócios, manipulando o armazenamento local, renderização da tabela e cálculo de tarifas. Os principais métodos incluem:

  • renderGarage(): Renderiza dinamicamente a tabela de veículos registrados.
  • checkOut(info): Calcula e exibe a tarifa e horário de saída ao registrar a saída de um veículo.
  • showAlert(message): Exibe mensagens de alerta para o usuário.
  • convertPeriod(startTime, endTime): Calcula o período de tempo entre o horário de entrada e saída de um veículo, convertendo-o em tarifa.

Conclusão

O sistema de estacionamento oferece uma solução simples e eficaz para gerenciar o fluxo de veículos em um estacionamento, proporcionando aos usuários a capacidade de registrar, monitorar e calcular as tarifas de forma automatizada. É projetado para ser intuitivo, fácil de usar e capaz de lidar com até 10 veículos simultaneamente.
